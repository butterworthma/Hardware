# Images Directory

## Required Images

Please place the following images in this directory:

### 1. Final Board Design - Top View
- **Filename**: `final_board_design_top.png` (or `.jpg`)
- **Description**: Top-down view of the complete BearWave board assembly
- **Content**: Blue PCB with green Raspberry Pi Zero 2 W, all components visible

### 2. Final Board Design - 3D View
- **Filename**: `final_board_design_3d.png` (or `.jpg`)
- **Description**: 3D rendering of the board showing component heights and layout
- **Content**: High-angle 3D view of the assembled board

### 3. PCB Layout - Top View
- **Filename**: `pcb_layout_top.png` (or `.jpg`)
- **Description**: Top view of the bare PCB before component assembly
- **Content**: Blue PCB with white silkscreen, showing component outlines

### 4. Schematic Diagram
- **Filename**: `schematic_diagram.png` (or `.pdf`)
- **Description**: Complete circuit schematic from EasyEDA
- **Content**: Full schematic showing all components and connections

## Image Specifications

- **Format**: PNG (preferred) or JPG
- **Resolution**: Minimum 1920x1080 for detailed views
- **Quality**: High resolution for component visibility
- **Naming**: Use descriptive filenames with underscores

## Usage

These images are referenced in:
- `README.md` - Main repository documentation
- `docs/board_annotations.md` - Detailed component breakdown
- `docs/schematic.md` - Circuit documentation
